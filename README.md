Heart Disease Prediction Project
Project Title
Intelligent Heart Disease Diagnostic Tool

Executive Summary
This project is about a computer program (Machine Learning) that helps doctors. It looks at patient data and predicts if a person has heart disease. My best model has an accuracy of 90.16%.

Problem Statement
Heart disease is a big health problem. It is hard for doctors to check every patient quickly. I created this tool to help identify high-risk patients faster using data.

Dataset Source
I used the Heart Disease Dataset. It has information from 303 patients, including their age, sex, cholesterol, and heart rate.

Methodology
I followed these simple steps:

Cleaning the Data: I made all the numbers a similar size (Scaling).

Testing Models: I tested 4 different ways to predict the disease:

K-Nearest Neighbors (KNN)

Support Vector Machine (SVM)

Logistic Regression

Random Forest

Choosing the Best: I compared the results to see which one is the most accurate.

Result
Here are my final accuracy scores:

K-Nearest Neighbors (KNN): 90.16% (My Best Model)

Support Vector Machine (SVM): 86.89%

Logistic Regression: 85.25%

Random Forest: 81.97%

Demonstration of the Application
You can use my application on the web.


How to use: Move the sliders to enter patient information and click the "Predict" button.
